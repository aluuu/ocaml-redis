String commands

val append : connection -> string -> string -> int S.IO.t
Append a value to a key; returns length of string after append.
val setbit : connection -> string -> int -> int -> int S.IO.t
Sets or clears the bit at offset in the string value stored at key.
val getbit : connection -> string -> int -> int S.IO.t
Returns the bit value at offset in the string value stored at key.
val bitop : connection ->
       bit_operation -> string -> string list -> int S.IO.t
Perform a bitwise operation between multiple keys (containing string values) and store the result in the destination key. See S.Client.bit_operation type for available operations.
val bitcount : ?first:int -> ?last:int -> connection -> string -> int S.IO.t
Count the number of set bits (population counting) in a string.
val bitpos : ?first:int -> ?last:int -> connection -> string -> int -> int S.IO.t
Return the position of the first bit set to 1 or 0 in a string.
val decr : connection -> string -> int S.IO.t
Decrements the number stored at key by one. If the key does not exist, it is set to 0 before performing the operation.
val decrby : connection -> string -> int -> int S.IO.t
Decrements the number stored at key by decrement. If the key does not exist, it is set to 0 before performing the operation.
val get : connection -> string -> string option S.IO.t
Get the value of key.
val getrange : connection -> string -> int -> int -> string option S.IO.t
Returns the substring of the string value stored at key, determined by the offsets start and end (both are inclusive).
val getset : connection -> string -> string -> string option S.IO.t
Atomically sets key to value and returns the old value stored at key. Returns None when key exists but does not hold a string value.
val incr : connection -> string -> int S.IO.t
Increments the number stored at key by one. If the key does not exist, it is set to 0 before performing the operation.
val incrby : connection -> string -> int -> int S.IO.t
Increments the number stored at key by increment. If the key does not exist, it is set to 0 before performing the operation.
val incrbyfloat : connection -> string -> float -> float S.IO.t
Increment the string representing a floating point number stored at key by the specified increment. If the key does not exist, it is set to 0 before performing the operation.
val mget : connection -> string list -> string option list S.IO.t
Returns the values of all specified keys.
val mset : connection -> (string * string) list -> unit S.IO.t
Sets the given keys to their respective values.
val msetnx : connection -> (string * string) list -> bool S.IO.t
Sets the given keys to their respective values. MSETNX will not perform any operation at all even if just a single key already exists.
val set : connection ->
       ?ex:int -> ?px:int -> ?nx:bool -> ?xx:bool -> string -> string -> bool S.IO.t
Set key to hold the string value.
val setex : connection -> string -> int -> string -> unit S.IO.t
Set key to hold the string value and set key to timeout after a given number of seconds.
val psetex : connection -> string -> int -> string -> unit S.IO.t
PSETEX works exactly like SETEX with the sole difference that the expire time is specified in milliseconds instead of seconds.
val setnx : connection -> string -> string -> bool S.IO.t
Set key to hold string value if key does not exist.
val setrange : connection -> string -> int -> string -> int S.IO.t
Overwrites part of the string stored at key, starting at the specified offset, for the entire length of value.
val strlen : connection -> string -> int S.IO.t
Returns the length of the string value stored at key. An error is returned when key holds a non-string value.
